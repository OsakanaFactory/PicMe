# フェーズ1 テスト仕様書 (MVP)

作成日: 2026/01/21
対象: PicMe フェーズ1 機能（認証、ダッシュボード、プロフィール、作品管理、SNSリンク、公開ページ）

## テスト環境
- Frontend: http://localhost:3001
- Backend: http://localhost:8080
- DB: PostgreSQL (Local)

## テストケース一覧

| ID | カテゴリ | テスト項目 | 手順 | 期待値 | 合否 |
|---|---|---|---|---|---|
| TC-001 | 認証 | 新規登録 | 1. `/signup` にアクセス<br>2. ユーザー名、Email、パスワードを入力して「登録」をクリック | ダッシュボード (`/dashboard`) にリダイレクトされること | |
| TC-002 | 認証 | ログアウト | 1. ダッシュボードのサイドバーから「ログアウト」をクリック | ログイン画面 (`/login`) にリダイレクトされること | |
| TC-003 | 認証 | ログイン | 1. `/login` にアクセス<br>2. 登録したEmail/パスワードでログイン | ダッシュボード (`/dashboard`) にリダイレクトされること | |
| TC-004 | プロフィール | プロフィール編集 | 1. `/profile` にアクセス<br>2. 表示名、自己紹介、画像をURLで入力して「保存」 | 「更新しました」等の成功メッセージが表示され、内容が保持されること | |
| TC-005 | プロフィール | 設定変更 | 1. テーマを「DARK」に変更して保存 | 画面全体のテーマがダークモードに切り替わること | |
| TC-006 | 作品管理 | 作品追加 | 1. `/artworks` にアクセス<br>2. 「新規追加」をクリック<br>3. タイトル、画像URLを入力して保存 | 一覧に新しい作品カードが表示されること | |
| TC-007 | 作品管理 | 作品削除 | 1. 追加した作品の「削除」ボタンをクリック<br>2. 確認ダイアログで「OK」 | 一覧から該当作品が消えること | |
| TC-008 | SNSリンク | リンク追加 | 1. `/social-links` にアクセス<br>2. 「リンクを追加」をクリック<br>3. プラットフォームとURLを入力して保存 | 一覧に新しいSNSリンクが表示されること | |
| TC-009 | 公開ページ | ページ表示 | 1. シークレットウィンドウ等で `http://localhost:3001/[ユーザー名]` にアクセス | プロフィール情報、作品一覧、SNSリンクが表示されること | |
| TC-010 | 公開ページ | テーマ確認 | 1. (TC-005実施後) 公開ページを確認 | ダークモードで表示されること | |
